rocket_launch Project Submission

Follow these steps to submit your hackathon project via GitHub Issues

schedule Submissions Open: December 2–8, 2025
1

how_to_reg Register Your Team

First, make sure your team is registered for the hackathon. Each team member must complete the registration form individually before December 1st.

open_in_new Register Here
2

folder_code Prepare Your Repository

Create a public GitHub repository for your project with the following:

  • code Complete source code with clear organization
  • description Detailed README.md with project description
  • install_desktop Setup and installation instructions
  • schema Architecture diagram (if applicable)
  • license License file (MIT recommended)

security Security Best Practices

Never commit API keys, passwords, or sensitive data. Use environment variables and include a .env.example file.

3

videocam Create Your Demo Video

Record a demo video (maximum 5 minutes) that includes:

  • play_circle A live demonstration of your solution
  • lightbulb Explanation of the problem you're solving
  • construction Description of your building experience
  • question_mark Why you chose this approach
  • smart_toy How AI helped you build this solution

Upload your video to YouTube (unlisted or public) or a similar platform.

4

category Choose Your Category

Select the category that best fits your project:

shield Threat Detection
security Cyber Intelligence
policy Organized Crime
smart_toy Generative & Agentic AI
gavel Governance & Public Policy
analytics Predictive Analytics
5

upload_file Submit via GitHub Issues

During the submission period (December 2-8), submit your project by creating a new GitHub Issue using our template. Your issue title should follow this format:

Project: <Your Project Title>, <Category>

The submission form will ask for:

  • badge Project Name: Short identifier for your project
  • article Description: Detailed description (treat as your README)
  • category Category: Your chosen challenge category
  • link Repository URL: Link to your GitHub repository
  • public Deployed Endpoint: (Optional) Live demo URL
  • movie Video URL: Link to your demo video
  • group Team Members: GitHub usernames of all team members

info Important

Confirm that all team members have individually completed the registration form. We cannot award prizes if team members aren't registered!

checklist Submission Checklist

check_circle All team members registered
check_circle GitHub repository is public
check_circle README.md is complete
check_circle Demo video under 5 minutes
check_circle Video is accessible
check_circle No sensitive data in repo
check_circle Category matches solution
check_circle Issue title format correct

Ready to Submit?

Submissions open December 2, 2025. Make sure you're registered before December 1st!
